Pattern
Allowed patterns:
  • abcd must match exactly abcd,
  • * matches anything (the same like empty pattern),
  • *abcd must end with abcd,
  • abcd* must start with abcd,
  • *abcd* must contains abcd,
  • ab*cd must start with ab and end with cd
